## C vs C++ Structs
| 特性      | C              | C++                          |
|---------|----------------|------------------------------|
| 成员初始化列表 | 不支持            | 支持                           |
| 成员函数    | 不支持            | 支持                           |
| 默认成员初始化 | 不支持            | 支持                           |
| 访问修饰符   | 不支持（默认为public） | 支持（public、private、protected） |

## C++ Struct vs Class
| 特性           | struct     | class       |
|--------------|------------|-------------|
| 默认的成员访问权限    | 公有（public） | 私有（private） |
| 成员和继承的默认访问权限 | 公有         | 私有          |
